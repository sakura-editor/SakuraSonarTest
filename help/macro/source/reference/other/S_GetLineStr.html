S_GetLineStr

機能
指定論理行の取得
構文
string S_GetLineStr ( i1 ) 使用例

S_GetLineStr の引数
引数型内容
i1整数数値行番号
設定値
引数 i1 の設定値は次のとおりです。

引数 i1 の設定値
値内容
0カーソルがある行を取得
1先頭行を取得
nn行目を取得
全行数より大きい値空文字列が返る
補足
PPAのマクロ中では $41 などの 16進数表記や変数・数式・関数の使用も可能だが、MACのマクロ中では 65 などの 10進数の定数表記しか受け付けない。
